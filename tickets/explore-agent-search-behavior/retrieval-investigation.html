⑤ Query rewrite — what exists today

  • Opt-in only via semfs grep --rewrite. OFF in every benchmark run — codex never passed it, so no rewrite occurred.
  • Model: OpenRouter openai/gpt-4.1-nano, temperature 0.0, one-shot. Fail-open to the original query on error.
  • Prompt: "Expand abbreviations, add closely-related terms and synonyms, keep it one concise line."
  • Does NOT translate — expands in the same language. That's why --rewrite still MISSED (more English ≠ Chinese content).
  • Fix idea: a translate-into-corpus-language rewrite (or append target-language terms) — the cheapest path to recover cross-lingual recall, no re-index needed.
